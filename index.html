What is Andrew?

Andrew is an NLP (Natural Language Processing) artificial intelligence that uses data from real Yu-Gi-Oh cards to take the user input of a name and attribute to return a description with stats of a unique A.I-generated monster.

Andrew

Our model Andrew uses our filtered data of every Yu-Gi-Oh card to generate a completely random card which can be used in a real Yu-Gi-Oh match!
